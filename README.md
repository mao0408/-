# 聊小智

独立桌面程序。打开后会贴在微信窗口旁边，识别当前聊天上下文，生成候选回复，支持编辑、发送、训练模式、自动识别和全托管。

## 使用方式

1. 启动程序。
2. 打开微信并点进某个具体聊天。
3. 点“识别当前聊天”，程序会截图并用本地 OCR 识别最近几条消息。
4. 程序生成 3 条候选回复，可点“改”编辑，也可点“发”直接发送。
5. 发送成功后，最终发送文本和当时上下文会写入长期记忆。

## 设置

首次启动可在右上角“设置”里填写：

- 智谱 API Key
- Base URL：`https://open.bigmodel.cn/api/paas/v4`
- 模型：`glm-4.7-flash`

设置保存到：

`D:\微信回复助手\user_data\settings.json`

## 导入聊天记录训练

设置窗口里有“导入聊天记录训练包”按钮。

训练包格式：

```text
A: 对方说的话
B: 你希望程序学习的回复
A: 对方下一句
B: 你的下一句
```

可以把多个 `.txt` 放进一个文件夹，然后右键文件夹选择“发送到 > 压缩(zipped)文件夹”，生成 `.zip` 后导入。

程序会保存 B 方回复前 5-8 轮完整上下文和真实回复，用于后续相似场景检索。长期记忆位置：

`D:\微信回复助手\user_data\memory.json`

## 打包给别人用

在项目目录执行：

```bat
python build_release.py
```

生成目录：

`dist\聊小智\`

把整个文件夹发给别人即可。对方解压后双击 `聊小智.exe` 或 `启动.bat`。

## 依赖

开发运行：

```bat
python -m pip install -r requirements.txt
python main.py
```

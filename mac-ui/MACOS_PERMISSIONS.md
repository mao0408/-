# macOS permissions

The assistant needs these permissions when the user enables OCR or sending:

- Screen Recording: capture the WeChat window for OCR.
- Accessibility: activate the chat window, click the input box, paste, and press Return.
- Clipboard access: read and restore the clipboard while sending.

These are user-granted macOS privacy permissions. The app can explain why a
permission is needed, but it cannot silently grant the permission. A real Mac
must be used for the final permission and WeChat smoke test.
